# SmileSync AI – Dental Appointment & AI Assistant Platform
SmileSync AI is an intelligent dental appointment management system that helps patients easily book dental appointments while assisting clinics in managing schedules efficiently. The platform also integrates an AI voice assistant to guide users and improve interaction.

🚀 Features
🦷 Appointment Booking System – Users can schedule appointments with dentists.
🔐 User Authentication – Secure login using Clerk authentication.
🗄 Database Management – PostgreSQL with Prisma ORM for efficient data handling.
🤖 AI Voice Assistant – Helps users with dental services and appointment booking.
📱 Responsive UI – Works smoothly across mobile, tablet, and desktop devices.
📊 Dashboard & Analytics – Admin dashboard with appointment and user statistics.
📧 Email Notifications– Automatic appointment confirmation emails.
📅 Appointment History – Users can view upcoming and past appointments.
⚙️ Dynamic UI Components– Modern interface with reusable components.

 🛠 Tech Stack

Frontend:React / Next.js
Backend:Node.js
Database:PostgreSQL
ORM:Prisma
Authentication:Clerk
AI Integration: OpenAI / AI Voice Assistant
Email Service: Nodemailer / SMTP

Installation
Clone the repository:

git clone https://github.com/yourusername/SmileSync-AI.git
Go to the project folder:
cd SmileSync-AI

Install dependencies:
npm install

Run the development server:
npm run dev

Open in browser:
http://localhost:3000

 🎯 Project Goal

The goal of SmileSync AI is to simplify dental clinic appointment management using automation, AI assistance, and a user-friendly interface.
